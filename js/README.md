# Kimberly Cloud Portfolio

A cloud-hosted portfolio website built using AWS and front-end web technologies.

## Technologies Used
- Amazon S3
- HTML
- CSS
- JavaScript

## Features
- Responsive portfolio website
- Visitor counter functionality
- Organized cloud deployment structure
- Static website hosting using Amazon S3
